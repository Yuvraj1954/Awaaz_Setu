========================================
AwaazSetu - Setup and Running Guide
========================================

PREREQUISITES:
- Python 3.7 or higher
- pip (Python package installer)

INSTALLATION STEPS:
-------------------

1. Navigate to the project directory:
   cd /path/to/AwaazSetu

2. Install required Python packages:
   pip install -r requirements.txt

   OR if you get an error about externally managed environment:
   pip install Flask flask-cors --break-system-packages

RUNNING THE APPLICATION:
------------------------

1. Start the Flask server:
   python3 app.py

   OR use the provided script:
   ./run.sh

2. Open your web browser and navigate to:
   http://localhost:5000

3. You should see the AwaazSetu interface

TROUBLESHOOTING:
----------------

Issue: "ModuleNotFoundError: No module named 'flask'"
Solution: Run: pip install Flask flask-cors

Issue: Port 5000 already in use
Solution: Either stop the other service using port 5000, or edit app.py
          and change the port number in the last line.

Issue: Cannot access from mobile device
Solution: When running Flask, note the IP address shown in terminal.
          Access from mobile using: http://[YOUR_IP]:5000
